# 🎮 GOODOQ ARCHIVE - Архив стримов Twitch

Полностью автоматизированный архив трансляций канала Twitch с восстановленным чатом.

## ⚡ Возможности

- 📹 Автоматическое скачивание всех VOD с канала
- 💬 Восстановление и синхронизация чата
- 🌐 Красивый веб-интерфейс
- 🔍 Поиск по стримам
- 📊 Статистика архива
- 🔄 Автоматическая синхронизация по расписанию

## 📋 Требования

- Python 3.9+
- Internet connection
- ~5GB свободного место (для видео)
